# Contractor Debrief — Copy-Paste This to Browser Agent

Copy everything below the line and paste it to the browser agent at the END of any session.
Works for any project, any platform.

---

## COPY FROM HERE ↓

Before you finish, I need you to do a complete debrief for my project manager. He cannot see what you see — he only reads your text output. Be thorough.

### 1. WHAT DID YOU CHANGE?
List every single change you made this session, in order:
- The exact page/URL where you made the change
- The field name
- What it was BEFORE (if you saw it)
- What you changed it TO
- Did it save successfully? (yes/no)

### 2. WHAT FAILED?
List anything that didn't work:
- What you tried to do
- The exact error message or behavior you saw
- What you think would fix it

If nothing failed, say "Nothing failed."

### 3. CURRENT STATE — SCREENSHOT EVERY PLATFORM
Go to each platform you touched this session. For each one:
- Take a screenshot
- Report the exact current status of everything you can see
- List any warnings, errors, or "action required" banners
- List any fields marked as "required" that are empty

### 4. WHAT'S BLOCKING?
Look at every platform dashboard. Find any checklist, progress bar, or "requirements" section.
- List every item shown
- Mark each as ✅ done or ❌ missing
- If there's a "ready to publish" or "ready to submit" indicator, report what it says

### 5. WHAT SHOULD HAPPEN NEXT?
Based on everything you can see right now, what needs to be done next? List the steps in order.

### FORMAT YOUR RESPONSE EXACTLY LIKE THIS:

```
DEBRIEF

Session date: [today's date]

Tasks completed: [number]
Tasks failed: [number]

(or "None")

[Platform name]:
- Status: [what you see]
- Warnings: [list or "none"]

BLOCKERS / REQUIREMENTS CHECKLIST:
- [item]: ✅ / ❌
- [item]: ✅ / ❌

NEXT STEPS:
1. [what needs to happen]
2. ...
```
