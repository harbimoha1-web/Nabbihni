# كم باقي — Screenshot Briefs (App Store + Play Store)

## Tool: AppMockUp Studio
**URL:** studio.app-mockup.com

| Setting | Value |
|---------|-------|
| Device (App Store) | iPhone 16 Pro Max (6.9") — required by Apple |
| Device (Play Store) | Samsung Galaxy S24 Ultra or Pixel 8 Pro |
| Frame Color | Black — matches app's dark theme |
| Export Format | PNG, highest resolution |
| Min Resolution | 1290×2796px |

**Workflow for each screenshot:**
1. Take screenshot on your phone/simulator
2. Upload to AppMockUp
3. Set background (specified per screen below)
4. Add headline text (top, centered, bold)
5. Add subtitle (below headline, smaller)
6. Export PNG
