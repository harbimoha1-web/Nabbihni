# Nabbihni Marketing Automation

Automated X/Twitter marketing for Nabbihni app using Make.com.

## Files

| File | Description |
|------|-------------|
| `X_AUTOMATION_SETUP.md` | Complete step-by-step setup guide |
| `fun_facts_database.csv` | 90 Arabic fun facts for all events |
| `events_calendar.csv` | Saudi events calendar (2026-2034) |
| `tweet_templates.md` | Ready-to-use tweet templates |
| `make_scenario_blueprint.json` | Make.com scenario reference |

## Quick Start

1. Read `X_AUTOMATION_SETUP.md` for full instructions
2. Create accounts: Make.com, X Developer
3. Create Google Sheet and import CSV files
4. Build scenario in Make.com
5. Test and activate

## Cost

**$0/month** - Uses free tiers only:
- Make.com: 1,000 ops/month (we use ~120)
- X API: Free tier
- Google Sheets: Free
