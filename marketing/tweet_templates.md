## Character Count Guide

| Element | Approx. Characters |
|---------|-------------------|
| Header + Days | 30-40 |
| Fact | 60-80 |
| CTA + Link | 50-60 |
| Hashtags | 25-35 |
| **Total** | **~200** (limit: 280) |

---

## Tips

1. **Vary templates** - Don't use the same template every day
2. **Match energy** - Use excited templates when close to event (< 7 days)
3. **Rotate facts** - Mark facts as "used" in the database
4. **Test length** - Ensure tweet fits within 280 characters
5. **Check hashtags** - Use trending Saudi hashtags when relevant
